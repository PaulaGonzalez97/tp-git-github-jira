## Materia

## Descripción
Este proyecto fue desarrollado como parte del trabajo práctico de gestión colaborativa utilizando Jira, Git y GitHub.

El trabajo consiste en la organización de tareas mediante Jira, el uso de control de versiones con Git y la documentación del proyecto en GitHub.

## Escenario elegido
Análisis de ventas de una pequeña empresa.

## Estructura del proyecto

- /datos → datasets utilizados
- /scripts → scripts de análisis
- /resultados → gráficos y resultados generados

## Tecnologías utilizadas

- Python
- Git
- GitHub
- Jira
- Google Colab
